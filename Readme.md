# Matchday Analytics

A Streamlit application for analysing football matches and seasons using event-level data.

---

## What this app shows

### Match view
- **Sequence event tracker**  
  Visual step-by-step tracking of actions on the pitch.
- **Shot map**  
  Shot locations for both teams, with goals highlighted.
- **Pass network**  
  Player-to-player passing connections based on successful passes.
- **Heatmap**  
  Event density by pitch zone for a selected team or both teams.
- **Progressive actions**  
  Progressive passes and carries based on pitch advancement.
- **Pressing view**  
  Defensive actions shown by pitch zones.
- **Final third & penalty area entries**  
  Passes entering advanced attacking areas.
- **Momentum chart**  
  Match control over time using action counts in fixed intervals.
- **Match report generator**  
  Generates a single static match report image including:
  - Scoreline  
  - Pass networks  
  - Shot map  
  - Momentum  
  - Key match statistics  

---

### Season view
- **Player rankings** across multiple metrics
- **Per-90 toggle** for fair comparison
- **Team-level analytics**
- **Interactive scatter charts**
